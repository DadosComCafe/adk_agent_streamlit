# ADK Agent Streamlit
Um repositório onde exploramos a criação de agentes de IA com o ADK, e o disponibilizamos em um servidor Streamlit!

## Como rodar
### 1. Clone o repositório:
__Para clonar o repositório, execute no terminal:__
    
    git clone https://github.com/DadosComCafe/adk_agent_streamlit

### 2. Instale as dependências:
__Para acessar a raíz do projeto clonado:__
    
    cd adk_agent_streamlit 

__Para sincronizar o ambiente com as dependências do projeto:__

    uv sync 

### 3. Rode o servidor streamlit:
Finalmente, rode o trecho abaixo. O navegador abrirá na página com o servidor streamlit

    uv run streamlit run main.py
